# Claude

## Overview
Claude is a series of large language models (LLMs) developed by [[company:anthropic]]. It is Anthropic's flagship product line, named in possible reference to mathematician Claude Shannon. Claude is fine-tuned using a technique called "constitutional AI," in which the model is trained to adhere to a set of principles called a constitution. The models are available via web interface, API, Amazon Bedrock, iOS app, and Mac and Windows desktop apps.

## Features
- Large language model capabilities (text generation, reasoning, coding, etc.)
- Constitutional AI fine-tuning for safety and alignment
- Available via web interface, API, Amazon Bedrock, iOS, Mac, and Windows desktop apps
- **Claude Code**: Command-line AI coding agent; reached general availability May 2025 with VS Code and JetBrains IDE integrations and GitHub Actions support
- **Claude Cowork**: Graphical UI equivalent of Claude Code, designed for simpler use
- **Web Search API**: Enables Claude to access real-time internet information (launched May 2025)
- **Claude Gov**: Specialized model for U.S. government and national security use (announced June 2025)
- **Model Context Protocol (MCP) connector**: New API capability introduced with Claude 4 (May 2025)

- Developers (API, Claude Code, MCP connector)

## Release History
- **March 2023**: Claude and Claude Instant released (initially to Anthropic-approved users only)
- **July 2023**: Claude 2 launched to the public
- **March 2024**: Claude 3 Opus, Claude 3 Sonnet, Claude 3 Haiku released (decreasing order of performance)
- **June 2024**: Claude 3.5 Sonnet released
- **May 2025**: Claude 4 released — Claude Opus 4 and Claude Sonnet 4, with improved coding capabilities
- **February 2026**: Claude Opus 4.6 released, followed by Sonnet 4.6
